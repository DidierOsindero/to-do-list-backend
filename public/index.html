To-Do list server API

RESTful JSON API

Retrieve all items

GET /to-dos

Create an item

POST /to-dos

Retrieve a specific item

GET /to-dos/{id}

Update a specific item

PATCH /to-dos/{id}

Attaching a JSON body with the possible below fields:

  • text (string)
  • complete (boolean)

Delete a specific item

DELETE /to-dos/{id}

Delete all completed to dos

DELETE /completed-to-dos
